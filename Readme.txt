It is an Ecommerce Website

Features: 
=> Implemented Session-based authentication strategy using passportJs
=> Implemented Redux strategy to manage Global state of application
=> Implemented Stripe webhook to manage payment flow in Backend
=> Followed MVC architecture for Backend.
=> Implemented the email functionality to Update users about their orders and account updates
=> Also added Invoice feature on successful Order of a Product.
=> Users can create or update their accounts, they can filter the product by category and brands also sort by price , rating.
=> They can also add or remove the product from the cart and checkout and use their desired payment method like cash or card to Order product.
=> There is pagination for products too.
=> Their is also admin login where he can edit or add the product and can also view all the orders that has been placed and can updated there status from pending,dispatched,delivered or cancelled.
